# Vue.js Application

Cette application utilise Vue.js avec Vite comme serveur de développement.

## Commandes

- `npm install` : Installe les dépendances.
- `npm run dev` : Démarre le serveur de développement.
- `npm run build` : Construit l'application pour la production.
- `npm run serve` : Prévisualise l'application en production.
